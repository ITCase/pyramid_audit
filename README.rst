Installing
----------

Install from github:

.. code-block:: bash

    pip install git+http://github.com/ITCase/pyramid_audit.git

PyPi:

.. code-block:: bash

    pip install pyramid_audit

Source:

.. code-block:: bash

    python setup.py install
